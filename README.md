Salt Bank - ATM GUI Simulator

A desktop banking application built with Python to demonstrate practical software development techniques, featuring a graphical user interface and persistent data storage. 

This project was developed as a technical showcase for my engineering SIWES portfolio.

 Features
Custom GUI: Built a retro-terminal interface using `Tkinter`.
* Persistent Storage: Reads and writes user profiles and balances using a `JSON` database.
* Authentication: Secure login system with PIN validation.
* Core Banking Logic: Handles deposits, withdrawals, transfers, and real-time transaction history tracking.
* Account Generation: Automatically creates unique 8-digit account numbers for new users.

Tech Stack
* Language: Python 3
* Libraries: `tkinter`, `json`, `datetime`, `random` (No external dependencies required)
